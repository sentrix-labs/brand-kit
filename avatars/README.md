# Profile Picture (PP / Avatar) Variants

Ready-to-use avatars for social platforms, in 8 style + color combinations.

## Variants

| Folder | Description |
|--------|-------------|
| `single-ring-black/` | Bronze ring + diamond on black bg — clean minimal |
| `single-ring-ivory/` | Bronze ring + diamond on ivory bg — light minimal |
| `single-ring-transparent/` | Bronze ring + diamond, transparent bg — overlay use |
| `solid-black/` | Big diamond on black bg, no ring — direct, bold |
| `solid-bronze-gold/` | Gold diamond on bronze bg — gold coin vibe, stands out |
| `solid-ivory/` | Bronze diamond on ivory bg — clean premium |
| `double-ring-black/` | Gold outer + bronze inner ring on black — coin-style |
| `double-ring-ivory/` | Gold outer + bronze inner ring on ivory — classy light |

## Available Sizes

Each variant is rendered in 10 sizes:

| Size | Platform / Use |
|------|----------------|
| 32×32 | Favicon small, compact icons |
| 64×64 | Favicon standard, small thumbnails |
| 128×128 | Chrome extension, small app icons |
| 180×180 | Apple iOS touch icon |
| 192×192 | Android Chrome icon |
| 256×256 | Trust Wallet, token lists |
| 400×400 | X/Twitter avatar, LinkedIn, Telegram min |
| 512×512 | Discord avatar, Android Play Store, Telegram recommended |
| 1024×1024 | iOS App Store, high-res display, X/Twitter max quality |
| 2048×2048 | Print, future-proof high-DPI |

## Quick Platform Guide

| Platform | Recommended File |
|----------|------------------|
| X (Twitter) PP | `avatar-{variant}-1024.png` |
| Discord | `avatar-{variant}-1024.png` |
| Telegram | `avatar-{variant}-512.png` |
| Instagram | `avatar-{variant}-1024.png` |
| GitHub org | `avatar-{variant}-512.png` |
| LinkedIn | `avatar-{variant}-400.png` |
| Favicon | `avatar-{variant}-32.png` + `-64.png` |
| iOS App Store | `avatar-{variant}-1024.png` |
| Android Play Store | `avatar-{variant}-512.png` |

Replace `{variant}` with the folder name (e.g., `solid-bronze-gold`, `double-ring-black`).

## Example paths

avatars/solid-bronze-gold/avatar-solid-bronze-gold-1024.png
