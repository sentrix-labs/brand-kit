# Premium X/Twitter Banners

10 background designs × 2 wordmark variants = **20 banner options** (1500×500).

## Naming pattern

- `x-banner-{style}-sentrix.png` — "SENTRIX" (shorter, larger logo)
- `x-banner-{style}-sentrix-chain.png` — "SENTRIX CHAIN" (formal name)

## When to use which wordmark

**"SENTRIX"** (shorter):
- Casual/community context
- Want the mark more prominent
- Discord, Telegram, general social

**"SENTRIX CHAIN"** (formal):
- First-time visitor banners
- Exchange listings, investor pages
- Disambiguation context
- X profile, LinkedIn

## Usage

1. Pick a style + wordmark variant
2. X → Edit Profile → Header → upload
3. 1500×500 exact match X spec, no cropping

PP-safe zone (left-bottom ~240×200px) clear on all banners.
